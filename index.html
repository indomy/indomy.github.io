Studi Kasus: Demo Ransomware

Sebenarkan konsep Ransomware itu ada karena tools enkripsi yang di gunakan untuk kepentingan aksi penculikan data, dan meminta tebusan untuk bisa mendekrip kembali data tersebut. Enkripsi itu alat yang netral, tergantung siapa yang pegang. Sama halnya golok di tangan petani dan tukang begal.

Disini kita akan coba pahami bagaimana Ransomware bekerja. Pertama kita coba untuk dahulu unduh ransomdemo.exe, coba taruh di desktop kita yang ada beberapa file kemudian jalankan. Maka akan terlihat semua file yang ada di desktop tersebut berubah ekstensinya menjadi .croot, itu pertanda file-file kita sudah terenkripsi.

Untuk Mengembalikan file kita bagaimana? gampang. Tinggal unduh aja dec.exe maka file-file kita yang terenkrip tadi akan jadi kembali seperti semua. Source code bisa diakses disini
